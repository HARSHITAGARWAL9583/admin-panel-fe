# Frontend

Minimal React + Vite frontend for Project Management App.

Set env variable:
- VITE_API_URL (e.g. https://your-railway-backend-url/api)

Install & Run:

```bash
cd frontend
npm install
npm run dev
```

Build for production:

```bash
npm run build
```

You can deploy the built static files to Railway Static (or any static host).
